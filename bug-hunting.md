50. **IronWASP** 🕷
   - An open-source web application security scanner that can be used to identify vulnerabilities in web applications.
51. **Wapiti** 🕷

52. **Reverse IP Lookup** 🔍
   - A tool that can be used to find all domains hosted on an IP address.
53. **DNS-Discovery** 🌐
   - A multithreaded subdomain bruteforcer that uses a word list to concatenate with a domain to look for subdomains.

59. **Katana** 🕷
   - A next-generation crawling and spidering framework.
60. **LinkFinder** 🔍

61. **JS-Scan** 🔍
   - A .js scanner, built in PHP, designed to scrape URLs and other info.
62. **LinksDumper** 🔍

63. **GoLinkFinder** 🔍

64. **BurpJSLinkFinder** 🔍

66. **HostileSubBruteforcer** 🔍

67. **second-order** 🔍

79. **flan** 🔍
   - A pretty sweet vulnerability scanner.
80. **Findsploit** 🔍

81. **BlackWidow** 🔍

82. **backslash-powered-scanner** 🔍

83. **Eagle** 🔍
